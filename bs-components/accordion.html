Accordion

Accordions are elements that expand or collapse when clicked on. They allow you to hide content that you do not want to consume page space unless the user decides that it is relevant to see.

Accordion Types

There are two accordion behaviors provided. To only allow one section to stay open at a time, specify data-accordion-type="accordion" on the <ul> element with the class="c-na-accordion". To allow multiple information panels to stay open, specify data-accordion-type="expandable" on the <ul> element with the class="c-na-accordion" attribute. See below for a demo of each type
