Separators

A user can choose which type of breadcrumb they want to show. The choices are /, |, ▸, ❯, or ≫

Divider Name Divider Parameter Divider Class
Slash Divider / .breadcrumbs__list--slash
Separator Divider | .breadcrumbs__list--separator
Arrow Divider ▸ .breadcrumbs__list--arrow
Greater Than Divider ❯ .breadcrumbs__list--greater
Double Greater Than Divider ≫ .breadcrumbs__list--double
