Loaders

Basic Loader

To include a loader create a div and add the class loader to that div. Be sure to add the class loader__wrapper to the parent div element.

Loader Loading Dots

To include a loader create a div and add the class loader__dots to that div. Be sure to add the class loader__wrapper to the parent div element.

Loader Stretch Bars

To include a loader create a div and add the class loader__bars to that div. Be sure to add the class loader__wrapper to the parent div element.For each bar create a div or span and add a class loader_bars(number) to the element such as loader_bars1
