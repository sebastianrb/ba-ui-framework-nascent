DropDown Alignment

You can position/align the dropdown menu to the top, right, left or bottom. by adding a class .dropdown__menu-show-(direction) to the ul element. You can also change the direction of the arrow of the dropdown toggle by adding the class .dropdown__arrow-(direction) to the span that contains the arrow. Below is a list of the possible classes that you can use to align the dropment and set the direction of the arrow. Also it is important to add the class dropdown__button-left to the button where you want to align the dropdown to the left

Arrow Parameter Arrow Description Dropdown Parameter DropDown Description
.dropdown__arrow--down Arrow points down .dropdown__menu The default state. The dropdown menu aligned to the bottom of the button toggle
.dropdown__arrow--left Arrow points left .dropdown__menu--show-left The dropdown menu will be aligned to the left of the toggle button
.dropdown__arrow--up Arrow points up .dropdown__menu--show-top The dropdown menu will be aligned to the top of the toggle button
.dropdown__arrow--right Arrow points right .dropdown__menu--show-right The dropdown menu will be aligned to the right of the toggle button
