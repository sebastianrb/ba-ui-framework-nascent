Innovative Pagination

This is an innovative pagination element. While it might look like a regular slider it is slightly different.

As a user moves the slider the numbers in the tooltip will change to reflect the position. A user can also enter the number on the tooltip and change the position or click the arrows in the tooltip to change the position. That is what makes this innovative because a user has three ways to change the page (slide the thumb, click the arrows or enter a number)

Create a form with two inputs. For the form add a class innovative-pagination__form. The first input should be an input[type=range]be sure to add the id sliderRange. The second input should be input[type=number] add the id inputNumber. Add min, max and value attributes to each element. All thre values should match for both inputs

in Firefox and IE the slider track color will change to show the progress of the slider.
